# BMICalculator
BMI 健康計算器 (C# WinForms)
這是一個使用 C# Windows Forms 開發的 BMI (體質指數) 計算工具，除了基礎的數值運算，還加入了直觀的顏色回饋與鍵盤快捷操作。

1.專案功能
-自動分類與配色：根據衛生福利部標準判斷「過輕、適中、過重及肥胖程度」，並根據結果改變顯示框背景顏色（例如：正常為綠色、過重為橘色、肥胖為紅色）。
-鍵盤快捷操作 (UX 優化)：
-在「身高」欄位按 Enter 自動跳轉至「體重」欄位。
-在「體重」欄位按 Enter 直接執行計算。
-按下 ESC 鍵可一鍵清除所有欄位並重新輸入。
-完善的防錯機制：自動偵測非數字輸入或負數，並彈出錯誤提示，確保程式不會崩潰。
2.執行說明
-開啟 1133516_bmi_hw1.sln 解決方案。
-使用 Visual Studio 編譯並執行。
-輸入身高 (cm) 與體重 (kg) 後點擊計算。
